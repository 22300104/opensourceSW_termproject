# 🎭 AR Face Filter Project - Sejoong

파이썬과 OpenCV, MediaPipe를 활용한 **증강현실(AR) 얼굴 필터 프로그램**입니다.  
얼굴을 인식하여 재미있는 액세서리(안경, 모자 등)를 씌우고, 손동작(제스처)으로 프로그램을 제어할 수 있습니다. 배경을 가상 이미지로 교체하는 기능도 지원합니다.

---

## 🚀 주요 기능 (Key Features)

### 1. 🧢 얼굴 필터 (Face Filters)
얼굴의 랜드마크를 정밀하게 추적하여 액세서리를 착용시킵니다.
- **안경 (Glasses)**: 눈 위치에 맞춰 안경을 씌워줍니다.
- **모자 (Hat)**: 머리 움직임에 따라 자연스럽게 따라다니는 모자입니다.
- **수염 (Mustache)**: 인중 위치에 수염을 붙여줍니다.
- **왕관 (Crown)**: 머리 위에 왕관을 씌워줍니다.
- **다중 착용**: 여러 필터를 동시에 착용할 수 있습니다.

### 2. 🖐 제스처 컨트롤 (Gesture Control)
키보드를 누르지 않고 손동작만으로 프로그램을 제어할 수 있습니다.
- **✌️ 브이 (V-Sign)**: 화면을 캡처하여 저장합니다. (스크린샷)
- **🖐️ 보 (Palm)**: 다음 필터 아이템으로 커서를 이동합니다.
- **✊ 주먹 (Fist)**: 현재 선택된 필터를 착용하거나 벗습니다.
- **☝️ 검지 (Index Finger)**: 가상 배경을 순서대로 변경합니다.

### 3. 🖼️ 가상 배경 (Virtual Background)
인물과 배경을 분리하여 배경만 다른 이미지로 교체합니다.
- **블러 모드**: 주변을 흐리게 처리하여 인물을 강조합니다.
- **컬러 배경**: 단색(핑크 등) 배경으로 교체합니다.
- **패턴 배경**: 별이 빛나는 밤하늘 배경 등으로 교체합니다.

### 4. 🎮 게임 같은 UI (Game-like UI)
- 화면 하단에 **인벤토리 바**를 제공하여 현재 선택된 아이템을 직관적으로 보여줍니다.
- **상태 메시지**: 제스처 인식이나 설정 변경 시 화면 중앙에 알림 텍스트가 뜹니다.

## 🛠️ 설치 및 실행 방법 (Installation)

### 1. 필수 라이브러리 설치
이 프로젝트는 Python 3.7 이상에서 동작합니다. 아래 명령어로 필요한 패키지를 설치해주세요.
```bash
pip install opencv-python mediapipe numpy pillow
```

### 2. 프로젝트 실행
터미널에서 `Sejung` 폴더로 이동한 뒤 `main.py`를 실행합니다.
```bash
cd Sejung
python main.py
```

## 🎮 조작 가이드 (Controls)

키보드와 제스처 두 가지 방식으로 조작할 수 있습니다.

### ⌨️ 키보드 조작 (Keyboard)
| 키 (Key) | 기능 (Function) |
| :---: | :--- |
| **A / D** | 아이템 커서 이동 (왼쪽 / 오른쪽) |
| **Space** | 아이템 착용 / 해제 (선택) |
| **Tab** | 가상 배경 모드 변경 |
| **S** | 스크린샷 저장 (Screenshots 폴더) |
| **R** | 동영상 녹화 시작 / 종료 (Videos 폴더) |
| **0** (숫자) | 모든 필터 및 배경 초기화 |
| **Q** | 프로그램 종료 |
| **+, -** | 필터 크기 조절 |
| **[, ]** | 필터 투명도 조절 |

### 🖐 손동작 조작 (Hand Gestures)
카메라에 손이 잘 보이도록 들어주세요.
* **✌️ (가위)**: 찰칵! 📸 (스크린샷)
* **🖐️ (보)**: 옆으로 이동 ➡️ (커서 이동)
* **✊ (주먹)**: 선택/해제 ✅ (아이템 착용)
* **☝️ (검지)**: 배경 변경 🖼️

## 💡 팁 (Tips)
* **제스처 인식**: 손을 얼굴 옆에 너무 가까이 대면 인식이 잘 안 될 수 있습니다. 얼굴과 겹치지 않게 어깨 높이 쯤 들어주세요.
* **조명**: 얼굴과 손이 밝게 나올수록 인식이 잘 됩니다. 역광을 피해주세요.
* **성능**: 컴퓨터 사양에 따라 가상 배경 기능 사용 시 렉이 발생할 수 있습니다.
